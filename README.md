# PassGen
Simple bash program that combines 6 words.


Installation:
git clone https://github.com/triple50/PassGen.git
cd PassGen
chmod +x PassGen
./PassGen

Usage:
Input like bellow
1st word
2nd word
3rd word
4th word
5th word
6th word
output file name
 
Future features:
Can cut any of the 6 words.
can put any other word,charachter beetween any two words.
